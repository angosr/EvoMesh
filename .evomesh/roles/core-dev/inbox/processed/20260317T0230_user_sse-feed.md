# 实现 SSE 实时消息流 — GET /api/feed/stream

## 功能
Server 端 SSE endpoint，实时推送角色动态到前端。

## 实现

```typescript
app.get("/api/feed/stream", (req, res) => {
  // Auth check
  res.setHeader("Content-Type", "text/event-stream");
  res.setHeader("Cache-Control", "no-cache");
  res.flushHeaders();

  // 每 5 秒检查角色 memory 变化
  const lastContent = new Map<string, string>();

  const check = () => {
    for (const p of projects) {
      for (const [name] of Object.entries(config.roles)) {
        const stm = readFile(memory/short-term.md);
        const key = `${p.slug}/${name}`;
        if (stm !== lastContent.get(key)) {
          lastContent.set(key, stm);
          // 提取最新一条 "Done" 内容
          const latest = extractLatestDone(stm);
          res.write(`data: ${JSON.stringify({
            type: "role-update",
            role: name,
            project: p.name,
            text: latest,
            time: new Date().toISOString()
          })}\n\n`);
        }
      }
    }
    // 检查 Central AI status 变化
    // 同理
  };

  check();
  const timer = setInterval(check, 5000);
  req.on("close", () => clearInterval(timer));
});
```
