## 消息渲染
每条消息一个 div，样式根据 type 区分：
- `role-update`: 左对齐，角色名彩色标签 + 时间 + 内容
- `central-status`: 左对齐，红色标签 "Central AI" + 内容
- `user-message`: 右对齐，蓝色背景
- `system`: 居中，灰色小字

```javascript
const es = new EventSource(`/api/feed/stream?token=${AUTH_TOKEN}`);
es.onmessage = (e) => {
  const msg = JSON.parse(e.data);
  appendFeedMessage(msg);
};
```

## 发送消息
```javascript
function sendFeedMsg() {
  const text = document.getElementById('feed-msg').value.trim();
  if (!text) return;
  appendFeedMessage({ type: 'user-message', text, time: new Date().toISOString() });
  authFetch('/api/admin/message', { method: 'POST', body: JSON.stringify({ message: text }) });
  document.getElementById('feed-msg').value = '';
}
```
