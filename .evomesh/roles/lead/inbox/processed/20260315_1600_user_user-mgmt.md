# 需求：用户管理系统设计

## 核心原则
- **默认不允许注册**，只有管理员可以增加用户
- 首次启动时创建的是管理员账户

## 需要设计的功能

### 1. 用户模型
- 角色：admin / viewer（admin 可管理用户和项目，viewer 只能查看）
- 存储：`~/.evomesh/users.yaml`（密码 hash 存储）
- 当前单用户密码系统需要升级为多用户

### 2. 管理员功能
- 添加用户（设置用户名、初始密码、角色）
- 删除用户
- 重置用户密码
- 在 Web UI 设置页面中管理

### 3. 暂不要求
- 不要求 2FA
- 不要求 session 过期
- 不要求权限细分到项目级别

请设计方案后分派给 executor 实现。
